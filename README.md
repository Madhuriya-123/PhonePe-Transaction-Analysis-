# 📊 PhonePe Transaction Analysis Dashboard | Power BI

An interactive **Power BI dashboard** built to analyze PhonePe transaction data and generate meaningful business insights. This project demonstrates end-to-end Business Intelligence skills, including data transformation, data modeling, DAX calculations, KPI reporting, and interactive dashboard development.

---

## 📌 Project Overview

The PhonePe Transaction Analysis Dashboard provides a comprehensive view of transaction performance, user behavior, payment success, and service-wise analysis. It helps stakeholders monitor key business metrics, identify trends, and make data-driven decisions through interactive visualizations.

---

## 🎯 Project Objectives

- Analyze overall transaction performance.
- Track Month-over-Month (MoM) growth.
- Compare Weekday vs Weekend transaction trends.
- Identify the highest-performing services.
- Analyze user distribution across age segments.
- Monitor payment success rate.
- Deliver actionable business insights through interactive dashboards.

---

## ❓ Business Questions Solved

- Which month recorded the highest number of transactions?
- Which service generated the highest transaction value?
- Which age segment contributed the most transactions?
- Are transactions higher on weekdays or weekends?
- What is the overall payment success rate?
- Which users generated the highest transaction value?
- How did transactions change over time?
- What are the Month-over-Month (MoM) growth trends?
